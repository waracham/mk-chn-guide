# mk-chn-guide

本项目用于归档和发布多个独立 HTML 页面，内容涵盖通用技术参考与资料整理。仓库本身不针对任何单个域名或具体网站，仅作为静态页面的集合与分发点。

## 项目简介

mk-chn-guide 是一个以 HTML 页面形式呈现的归档仓库。所有页面均为独立文件，可直接在浏览器中打开，无需构建或服务端支持。适合用于资料查阅、离线浏览或作为小型知识库使用。

## 目录说明

```
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的静态资源（样式、图片、脚本等）
├── index.html      # 仓库主页（可选，提供页面索引）
└── README.md       # 本文件
```

- `pages/` 目录为核心内容，每个 HTML 文件对应一个独立主题或资料页。
- `assets/` 目录存放页面所需的公共资源，按需组织。
- `index.html` 为可选的导航页，用于快速浏览所有页面。

## 页面归档说明

- 所有页面均为纯 HTML，不依赖外部数据库或后端服务。
- 页面之间无强制关联，可单独打开使用。
- 内容以技术参考、使用指南、资料整理为主，不包含商业推广或特定网站链接。
- 页面编码统一为 UTF-8，兼容常见浏览器。

## 维护说明

- 欢迎提交 Pull Request 增加新的页面或修正现有内容。
- 页面应保持独立、简洁，避免引入过多外部依赖。
- 新增页面请在 `pages/` 下创建，并在 `index.html`（如有）中添加链接。
- 请勿在页面中嵌入任何形式的广告、跟踪代码或诱导性内容。

## 使用方式

1. 克隆仓库到本地：
   ```bash
   git clone https://github.com/your-username/mk-chn-guide.git
   ```
2. 直接用浏览器打开 `index.html` 或 `pages/` 下的任意 HTML 文件。
3. 也可通过 GitHub Pages 等静态托管服务在线访问。
